Level 3 — Sub-Task A

Above-Average Risk Conditions

Identifies conditions where accident counts exceed the statewide per-condition average. Uses a nested SQL query — the outer query filters on results produced by an inner average calculation — to discover new facts beyond simple totals.

Nested query logic
SELECT condition, total, avg_severity
FROM (inner summary per condition)
WHERE total > (SELECT AVG(count) FROM (grouped counts))
ORDER BY avg_severity ASC
